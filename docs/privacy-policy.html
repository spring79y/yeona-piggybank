내 아이의 저금통

개인정보 처리방침 · 최종 업데이트: 2026년 6월 28일

「내 아이의 저금통」(이하 「앱」)은 사용자의 개인정보를 소중히 여깁니다. 본 방침은 앱이 어떤 정보를 다루는지 설명합니다.

요약
  • 회원가입·로그인 없음
  • 개인정보를 서버로 전송하거나 수집하지 않음
  • 모든 데이터는 사용자 기기에만 저장
  • 광고·분석·추적 SDK 미사용

1. 수집하는 정보

앱은 사용자 개인정보를 수집하지 않습니다.

다음 정보는 사용자가 직접 입력하며, 기기 내부에만 저장됩니다.

  • 엄마 비밀번호 (기기 로컬 저장)
  • 아이 이름 (선택 입력, 기기 로컬 저장)
  • 저금통 잔액·거래 기록
  • 할 일 목록 및 완료 상태
  • 칭찬스티커·돌림판 상태
  • 앱 설정 (투자 수익률, 알림 on/off 등)

위 정보는 개발자 서버로 전송되지 않습니다.

2. 정보 저장 방식

  • iOS 기기의 UserDefaults 및 App Group 저장소 사용
  • 앱과 홈 화면 위젯 간 데이터 공유를 위해 App Group(group.com.yeona.piggybank) 사용
  • 앱 삭제 시 해당 데이터도 함께 삭제됩니다

3. 알림

할 일 미완료 알림을 사용하려면 iOS 알림 권한이 필요합니다. 알림은 기기에서만 예약되며, 외부 서버를 거치지 않습니다. 알림을 끄거나 iOS 설정에서 권한을 철회할 수 있습니다.

4. 제3자 제공 및 공유

앱은 수집한 정보를 제3자에게 제공·판매·공유하지 않습니다. 광고 네트워크, 분석 도구, 소셜 로그인 등을 사용하지 않습니다.

5. 아동 개인정보

앱은 아이와 부모가 함께 사용하도록 설계되었습니다. 아이 이름은 사용자(부모)가 선택적으로 입력하며, 기기에만 저장됩니다. 13세 미만 아동으로부터 개발자가 직접 개인정보를 수집하지 않습니다.

6. 보안

입금·출금·설정 변경 등은 엄마 비밀번호로 보호됩니다. 비밀번호는 기기에만 저장되며, 개발자가 접근할 수 없습니다.

7. 사용자 권리

  • 앱 내 「엄마 관리 → 전체 초기화」로 모든 로컬 데이터 삭제 가능
  • 앱 삭제 시 기기에 저장된 모든 데이터 삭제

8. 방침 변경

본 방침이 변경되면 이 페이지의 「최종 업데이트」 날짜를 수정합니다. 중요한 변경이 있을 경우 App Store 업데이트 설명을 통해 안내할 수 있습니다.
