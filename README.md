# 연아의 저금통

Give, Spend, Save 세 가지 저금통으로 용돈을 나누어 관리하는 iPad 앱입니다.

## 기능

- **메인 화면**: Give(나누기), Spend(쓰기), Save(모으기) 세 개의 저금통
- **잔액 표시**: 각 저금통에 현재 적립 금액(원) 표시
- **채움 표현**: 입금액에 따라 저금통이 차오르는 시각적 표현 (목표 100,000원 기준)
- **입금/출금**: 저금통 터치 → 상세 화면에서 금액 입력 및 빠른 금액 버튼(1,000 / 5,000 / 10,000원)
- **데이터 저장**: 앱을 껐다 켜도 잔액 유지

## 실행 방법 (Xcode 네이티브 앱)

1. Mac에 Xcode 설치
2. `YeonaPiggyBank.xcodeproj` 더블클릭으로 열기
3. 상단에서 **iPhone/iPad** 시뮬레이터 또는 실제 기기 선택
4. ▶️ Run 버튼으로 빌드 및 실행
5. 실제 기기에 설치하려면 Apple Developer 계정으로 서명 설정 필요

## 프로젝트 구조

```
YeonaPiggyBank/
├── YeonaPiggyBank.xcodeproj/   # Xcode 프로젝트
├── YeonaPiggyBank/             # SwiftUI 소스
│   ├── Models/                 # 데이터 모델
│   ├── Views/                  # 화면 UI
│   └── Helpers/                # 헬퍼·설정
└── YeonaPiggyBankWidget/       # 홈 화면 위젯
```

## 저금통 설명

| 이름  | 한글   | 용도           | 색상   |
|-------|--------|----------------|--------|
| Give  | 나누기 | 기부·선물용    | 핑크   |
| Spend | 쓰기   | 소비·구매용    | 오렌지 |
| Save  | 모으기 | 저축·목표용    | 민트   |
